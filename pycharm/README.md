Config file location (referenced as `CONFIG_DIR` in other READMEs):

* Before 2020: `$HOME/.PyCharmCE----/config`
* From 2020: `$HOME/.config/JetBrains/PyCharmCE---`

**NB:** `----` is the version of your PyCharm IDE
